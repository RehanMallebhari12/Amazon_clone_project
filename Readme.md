The Amazon Clone (Frontend Only) project is a web-based application designed to replicate the user 
interface and core functionalities of Amazon’s e-commerce website using HTML, CSS, and 
JavaScript. The project aims to provide a visually appealing and user-friendly shopping experience by 
implementing essential features such as product listing, cart management, and a checkout page. 
This project is static and frontend-focused, meaning it does not include backend features like user 
authentication or payment processing. The website layout will be responsive and mobile-friendly, 
ensuring compatibility with different screen sizes.

Modules :
The project is divided into the following main modules: 
1. Header & Navigation Bar  
Amazon-style navigation bar with search bar, cart icon, and user account dropdown. 
2. Home Page 
Hero image slider with promotional banners. 
Category-wise product listing using a grid layout. 
3. Product Page 
Product details page with images, description, price, and add-to-cart button.
4. Cart Page 
List of added products, quantity adjustment, and remove item option. 
Total price calculation dynamically using JavaScript. 
5. Checkout Page 
User form to enter shipping details. 
Order summary displaying selected products and total cost. 
6. Styling & Responsiveness 
Using CSS Flexbox & Grid for layout. 
Media queries to make the website mobile-friendly.
7. Footer Section
Includes useful navigation links, customer support links, company information, and additional resources similar to the Amazon website.
